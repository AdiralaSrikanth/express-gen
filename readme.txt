Express generator, generates boiler plate for your project with public, views and routes etc folders

1. To install express generator type in command in terminal
npm install -g express-generator

2. Create express app by using the following command, Type in express then set view prop as pug/ejs/hbs and then application name 
 express --view=ejs myapp

3. we have dependencies but not installed in our local, to generate node_modules type in the project
npm i
